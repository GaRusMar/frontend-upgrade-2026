# Day 03 — Codex

## Что можно отдать Codex

Codex можно попросить:

- создать структуру папки урока;
- создать `index.html`;
- создать `style.css`;
- создать пустой `script.js` с TODO;
- проверить мой код после самостоятельной реализации;
- объяснить ошибку из консоли;
- объяснить разницу между `map`, `filter`, `find`, `forEach`;
- проверить, не мутирую ли я массив через `sort`;
- предложить рефакторинг после того, как я сам написал решение.

## Что студент пишет сам

Самостоятельно пишу:

- массив `products`;
- массив `categories`;
- массив `cartItems`;
- Function Declaration;
- Function Expression;
- Arrow Function;
- callback-функции;
- `forEach`;
- `map`;
- `filter`;
- `find`;
- `sort`;
- расчёт cart total;
- функцию с замыканием;
- вывод данных на страницу.

## Что нельзя отдавать Codex

Codex нельзя просить:

- полностью написать `script.js`;
- написать все функции за меня;
- написать все методы массивов за меня;
- исправить код без объяснения причины;
- заменить моё понимание готовым решением;
- коммитить код, который я не могу объяснить.
