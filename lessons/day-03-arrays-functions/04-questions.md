## Ответы

1. Массив — это список значений.
2. С нуля.
3. Количество элементов массива.
4. Добавляет элемент в конец массива.
5. Удаляет последний элемент массива и возвращает его.
6. Это массив, где каждый элемент является объектом.
7. Функция — блок кода, который можно вызвать.
8. Возвращает результат из функции.
9. Функция вернёт `undefined`.
10. Объявление функции через `function name() {}`.
11. Функция, записанная в переменную.
12. Короткий синтаксис функции через `=>`.
13. Функция, переданная в другую функцию.
14. Чтобы выполнить действие или проверку для каждого элемента.
15. Выполняет действие для каждого элемента массива.
16. `undefined`.
17. Создаёт новый массив на основе старого.
18. `map` возвращает новый массив, `forEach` просто выполняет действие.
19. Создаёт новый массив элементов, прошедших проверку.
20. Возвращает первый элемент, прошедший проверку.
21. `filter` возвращает массив, `find` возвращает один элемент или `undefined`.
22. Сортирует массив.
23. Потому что `sort` изменяет исходный массив.
24. Метод для сравнения строк при сортировке.
25. Пройти по массиву и сложить `price * quantity`.
26. Когда функция помнит переменные из внешней функции.
27. Для создания заранее настроенных фильтров, например по категории.
28. Если `find` ничего не нашёл, результат `undefined`, и обращение к свойствам даст ошибку.
29. В списке товаров, категорий, корзине, заказах.
30. В форматировании цены, фильтрации, поиске, сортировке, расчётах.
