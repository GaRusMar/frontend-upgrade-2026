## Что делаем

Создаём мини-калькулятор для будущего проекта **Bangkok Express**.

Практика:

```text
Price / Delivery Calculator
```

Пользователь вводит:

```text
Dish price
Quantity
Delivery price
Promo code
```

Приложение считает:

```text
Items total
Delivery
Discount
Final total
```

## Логика

Правила расчёта:

```text
itemsTotal = dishPrice * quantity

если promo code === "BANGKOK10":
  discount = itemsTotal * 0.1

если itemsTotal >= 30:
  delivery = 0
иначе:
  delivery = deliveryPrice

finalTotal = itemsTotal - discount + delivery
```

## Почему это важно для frontend

Любой frontend-интерфейс работает по похожему сценарию:

```text
пользователь вводит данные
→ JavaScript читает данные
→ JavaScript проверяет данные
→ JavaScript считает результат
→ JavaScript обновляет интерфейс
```

В Bangkok Express такая логика понадобится для:

```text
- расчёта цены блюда
- количества товаров
- скидок
- доставки
- итоговой суммы заказа
- корзины
- формы заказа
```

## Критерий готовности

Задача выполнена, если:

- значения читаются из input;
- строки преобразуются в числа;
- пустые или неверные значения показывают ошибку;
- количество меньше 1 не принимается;
- промокод `BANGKOK10` даёт скидку 10%;
- при сумме товаров от 30 доставка становится бесплатной;
- итоговая сумма считается правильно;
- результат выводится на страницу;
- ошибки видны пользователю;
- в консоли нет неожиданных ошибок;
- студент может объяснить каждую строку своего JavaScript-кода.
