# Day 13 Theory: password generator

## Ключевые понятия урока
- DOM-элементы и их поиск
- События и обработчики
- Состояние интерфейса
- Обновление UI

## Что нужно понять перед практикой
- Какие данные являются состоянием
- Где и как изменяется состояние
- Как синхронизировать состояние и интерфейс

## Частые ошибки
1. Смешивать логику состояния и разметку без структуры.
2. Дублировать код в нескольких обработчиках.
3. Забывать обновлять UI после изменения состояния.
4. Игнорировать проверку на null при поиске элементов.
