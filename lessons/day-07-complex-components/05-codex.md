# Day 07 — Codex

## Что можно отдать Codex

Codex можно попросить создать HTML/CSS, пустой `script.js` с TODO, проверить код после самостоятельной реализации, объяснить ошибку из консоли, проверить modal, overlay click, Escape close, autocomplete keyboard navigation, debounce/throttle, edge cases, предложить рефакторинг в классы после самостоятельной реализации.

## Что студент пишет сам

Самостоятельно пишу modal open/close, overlay click close, Escape close, ribbon menu, active category, spicy slider, autocomplete, suggestions render, keyboard navigation, debounce, throttle, drag'n'drop preview, edge case проверки.

## Что нельзя отдавать Codex

Codex нельзя просить полностью написать `script.js`, написать все компоненты за меня, написать debounce/throttle без понимания, сразу превращать всё в классы без самостоятельной реализации, коммитить код, который я не могу объяснить.

## Workflow

```text
1. Прочитать 01-task.md
2. Прочитать 02-theory.md
3. Выполнить 03-practice.md
4. Написать JavaScript самостоятельно
5. Проверить modal / ribbon / slider / autocomplete / drag / scroll
6. Проверить edge cases
7. Ответить на вопросы из 04-questions.md
8. Только после этого просить Codex сделать review
9. Сделать коммит
```
