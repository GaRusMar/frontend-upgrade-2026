# Day 08 — Codex

## Что можно отдать Codex

Codex можно попросить создать HTML/CSS, пустой `script.js` с TODO, пример `products.json`, проверить код после самостоятельной реализации, объяснить ошибку fetch/FormData/regexp/CORS, проверить loading/error/success states, проверить disabled submit и то, что форма не очищается при ошибке.

## Что студент пишет сам

Самостоятельно пишу submit handler, preventDefault, FormData, trim, validation, regexp, errors object, render errors, fetch GET, response.ok, response.json, render products, fetch POST, Content-Type, JSON.stringify, isSubmitting, disabled submit button, loading/error/success states.

## Что нельзя отдавать Codex

Codex нельзя просить полностью написать `script.js`, написать всю валидацию или fetch-flow за меня, скрыть ошибки без объяснения, отправлять реальные данные на неизвестный сервер, коммитить код, который я не могу объяснить.
