# Day 01 Codex Rules

## Что можно отдать Codex
- Создание структуры файлов.
- Подготовку HTML-разметки.
- Подготовку базовых CSS-стилей.
- TODO-шаблон для script.js.
- Ревью и объяснение ошибок после твоей реализации.

## Что нельзя отдавать Codex
- Основную JavaScript-логику счётчика.
- Готовые обработчики +1, -1, Reset.
- Готовую реализацию updateCounter.

## Правильный workflow
1. Codex подготавливает структуру и каркас.
2. Ты сам пишешь JS-логику.
3. Проверяешь результат в браузере.
4. Просишь Codex сделать ревью и подсветить ошибки.
5. Исправляешь и коммитишь.
