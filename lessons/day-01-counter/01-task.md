# Day 01: Counter

## Цель урока
Понять базовый цикл frontend-логики: действие пользователя -> изменение состояния -> обновление интерфейса.

## Что делаем
Собираем счётчик с кнопками +1, -1, Reset.

## Какие темы закрепляем
- const
- let
- querySelector
- addEventListener
- count как состояние
- textContent
- classList
- updateCounter

## Критерий готовности
- +1, -1, Reset работают.
- Значение на экране всегда соответствует count.
- Цвет/класс значения меняется по знаку числа.
- В консоли нет ошибок.

## Итог урока
Я умею хранить состояние, обрабатывать события и обновлять UI через отдельную функцию.
