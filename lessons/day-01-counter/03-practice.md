# Day 01 Practice: Counter

## Шаг 1: найти элементы
Найди кнопки +1, -1, Reset и элемент со значением счётчика.

## Шаг 2: создать count
Создай состояние: let count = 0.

## Шаг 3: сделать кнопку +1
По клику увеличивай count на 1.

## Шаг 4: сделать кнопку -1
По клику уменьшай count на 1.

## Шаг 5: сделать Reset
По клику устанавливай count = 0.

## Шаг 6: вынести updateCounter
Сделай функцию updateCounter, которая обновляет интерфейс после каждого изменения состояния.

## Шаг 7: добавить цвета
Добавь переключение классов для положительного, отрицательного и нулевого значения.

## Шаг 8: проверить в браузере
Проверь все кнопки и отсутствие ошибок в консоли.
