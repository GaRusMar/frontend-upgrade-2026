# Course Project — Bangkok Express

## Цель проекта

Собрать финальный проект по блоку `jsbasic`.

Проект должен объединить всё, что было пройдено в Day 01–10:

```text
- основы JavaScript
- объекты
- массивы
- функции
- DOM
- события
- делегирование
- компоненты
- ООП
- CustomEvent
- modal
- ribbon menu
- carousel
- slider
- формы
- validation
- fetch
- JSON
- Promise
- async/await
- TypeScript architecture notes
```

## Что делаем

Финальный проект:

```text
Bangkok Express
```

Это учебный интерфейс ресторана с онлайн-заказом.

## Что должно быть в проекте

```text
- карусель товаров сверху
- ленточное меню категорий
- слайдер / фильтр
- подгрузка товаров
- динамическая фильтрация
- список товаров
- корзина
- модальное окно корзины
- форма заказа
- валидация формы
- loading / error / success states
- async flow через Promise / async-await
```

## Порядок прохождения

1. Открой `01-requirements.md`
2. Прочитай `02-architecture.md`
3. Прочитай `03-components.md`
4. Прочитай `04-data-and-api.md`
5. Выполни задачи из `05-tasks.md`
6. Проверь проект по `06-acceptance-criteria.md`
7. Ответь на `07-interview-questions.md`
8. Проверь `08-codex.md`
9. Реализуй JavaScript самостоятельно
10. Проверь проект в браузере
11. Сделай финальный коммит

## Главное правило

HTML и CSS можно подготовить заранее.

Основную JavaScript-логику студент пишет сам.

Codex можно использовать для структуры, review, объяснения ошибок и коммитов.

Codex нельзя использовать как замену собственного понимания.
