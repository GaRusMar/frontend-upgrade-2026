# Day 09 — Codex

## Что можно отдать Codex

Codex можно попросить создать HTML/CSS, пустой `script.js` с TODO, проверить код после самостоятельной реализации, объяснить event loop, Promise chain, Promise methods и async/await.

## Что студент пишет сам

Самостоятельно пишу fakeRequest, resolve/reject сценарии, Promise chain, then/catch/finally, Promise.all, race, allSettled, callback API, promisification, async/await, try/catch/finally, loading/error/success state.

## Что нельзя отдавать Codex

Codex нельзя просить полностью написать `script.js`, написать все Promise-сценарии за меня, заменить понимание event loop готовым решением, коммитить код, который я не могу объяснить.
