# Day 03 — Массивы и Функции

## Ориентир по программе jsbasic

Тема из программы:

> Массивы и Функции

В этом уроке разбираем способы задания функций в JavaScript:

```text
- Function Declaration
- Function Expression
- функции-стрелки
- их отличия
```

Также изучаем:

```text
- замыкания
- callback-функции
- методы массивов, которые используют callback
- сортировка
- поиск
```

По окончанию этого занятия ты должен знать JavaScript и уметь работать с основными структурами данных в достаточной степени, чтобы перейти к созданию интерфейсов.

## Тема дня

```text
массивы → функции → callback → методы массивов → поиск → сортировка
```

## Цель урока

После урока ты должен уверенно понимать:

- что такое массив;
- зачем массивы нужны во frontend;
- как создать массив объектов;
- как получить элемент массива по индексу;
- как узнать длину массива;
- как добавить элемент в массив;
- как удалить элемент из массива;
- чем отличаются Function Declaration, Function Expression и Arrow Function;
- что такое callback;
- почему методы массивов принимают callback-функции;
- как использовать `forEach`;
- как использовать `map`;
- как использовать `filter`;
- как использовать `find`;
- как использовать `sort`;
- как считать total по массиву;
- что такое простое замыкание на практике.

## Что делаем

Создаём набор функций для работы с меню будущего проекта **Bangkok Express**.

Практика:

```text
Bangkok Express Products Analyzer
```

Ты будешь работать с массивами:

```text
products
categories
cartItems
```

## Что должен уметь код

В `script.js` нужно самостоятельно описать логику:

```text
1. Создать массив products
2. Создать массив categories
3. Показать количество товаров
4. Найти первый товар
5. Добавить новый товар
6. Удалить последний товар
7. Отрисовать названия товаров
8. Получить массив названий через map
9. Отфильтровать товары по категории
10. Найти товар по id
11. Найти товар по названию
12. Отсортировать товары по цене
13. Посчитать среднюю цену
14. Посчитать total корзины
15. Создать функцию-фильтр через замыкание
16. Вывести результаты на страницу
```

## Почему это важно для frontend

Во frontend почти всегда приходится работать со списками:

```text
- список товаров
- список категорий
- список пользователей
- список заказов
- список задач
- список сообщений
```

В Bangkok Express массивами будут:

```text
- products[]
- categories[]
- cartItems[]
```

Если ты хорошо понимаешь массивы и функции, дальше будет проще с:

```text
- генерацией карточек товаров
- фильтрами
- поиском
- сортировкой
- корзиной
- API-ответами
- React render list
```

## Критерий готовности

Задача выполнена, если:

- массив `products` создан самостоятельно;
- каждый product — объект;
- используются `push` и `pop`;
- есть минимум 3 вида функций;
- есть пример callback-функции;
- используется `forEach`;
- используется `map`;
- используется `filter`;
- используется `find`;
- используется `sort`;
- есть расчёт total корзины;
- есть пример простого замыкания;
- данные выводятся на страницу;
- в консоли нет неожиданных ошибок;
- студент может объяснить каждую строку своего JavaScript-кода.
