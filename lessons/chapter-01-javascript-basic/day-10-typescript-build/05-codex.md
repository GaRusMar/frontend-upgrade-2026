# Day 10 — Codex

## Что можно отдать Codex

Codex можно попросить создать структуру папок, пустые файлы, базовый package.json, tsconfig.json, проверить TypeScript после самостоятельной реализации, объяснить ошибки typecheck/build.

## Что студент пишет сам

Самостоятельно пишу Product type, Category union, CartItem interface, Order interface, products data, formatPrice, ProductCard, Cart, imports/exports, main.ts.

## Что нельзя отдавать Codex

Codex нельзя просить полностью написать все TypeScript-файлы, типизировать весь проект без понимания, ставить any везде, коммитить код, который я не могу объяснить.
