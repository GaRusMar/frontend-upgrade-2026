# Day 05 — Codex

## Что можно отдать Codex

Codex можно попросить создать HTML/CSS, пустой `script.js` с TODO, проверить мой код после самостоятельной реализации, объяснить ошибку из консоли, объяснить `target/currentTarget`, проверить делегирование событий.

## Что студент пишет сам

Самостоятельно пишу обработчики событий, `event`, `target/currentTarget`, делегирование, `closest`, чтение `dataset`, `data-action`, active-класс, фильтрацию, Add to cart, cart counter, carousel next/prev, event log.

## Что нельзя отдавать Codex

Codex нельзя просить полностью написать `script.js`, написать всю логику событий за меня, исправить код без объяснения причины, коммитить код, который я не могу объяснить.

## Workflow

```text
1. Прочитать 01-task.md
2. Прочитать 02-theory.md
3. Выполнить 03-practice.md
4. Написать JavaScript самостоятельно
5. Проверить клики в браузере
6. Проверить DOM во вкладке Elements
7. Проверить ошибки во вкладке Console
8. Ответить на вопросы из 04-questions.md
9. Только после этого просить Codex сделать review
10. Сделать коммит
```
