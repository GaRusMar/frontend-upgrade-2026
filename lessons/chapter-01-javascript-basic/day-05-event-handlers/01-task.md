## Ориентир по программе jsbasic

Тема из программы:

> Обработчики событий

Изучаем взаимодействие с посетителем при помощи событий.

В этом уроке разбираем:

```text
- установка обработчиков событий
- свойства событий
- погружение
- всплытие
- делегирование событий
- работа с большим количеством элементов
- упрощение сложных интерфейсов
- поведение через HTML-атрибуты
```

## Тема дня

```text
события → event object → target/currentTarget → bubbling → delegation → data-action
```

- чем `event.target` отличается от `event.currentTarget`;
- зачем нужен `preventDefault`;
- как работает всплытие событий;
- что такое погружение событий;
- что такое делегирование событий;
- зачем использовать `data-action`;
- как управлять интерфейсом через HTML-атрибуты;
- как отлаживать события в DevTools;
- как сделать меню, карусель и добавление товара в корзину.

## Что делаем

Создаём интерактивный экран для будущего проекта **Bangkok Express**.

Практика:

```text
Bangkok Express Events Playground
```

Интерфейс содержит:

```text
- меню категорий
- список карточек товаров
- кнопку Add to cart
- счётчик корзины
- простую карусель изображений
- панель логов событий
```

## Что должен уметь код

```text
1. Найти DOM-элементы
2. Повесить обработчик на меню категорий
3. Переключать active-категорию
4. Фильтровать товары по категории
5. Использовать делегирование на списке товаров
6. Обрабатывать клик по Add to cart
7. Читать product id из dataset
8. Обновлять cart counter
9. Сделать карусель: next / prev
10. Использовать data-action для кнопок
11. Логировать события в event log
12. Проверить target и currentTarget
```

## Почему это важно для frontend

Frontend — это взаимодействие с пользователем.

В Bangkok Express события понадобятся для кликов по категориям, Add to cart, открытия корзины, закрытия модального окна, переключения карусели, фильтров и формы заказа.

## Критерий готовности

Задача выполнена, если:

- категории переключаются по клику;
- active-класс переносится на выбранную категорию;
- товары фильтруются по категории;
- Add to cart работает через делегирование;
- id товара читается через `dataset`;
- cart counter увеличивается;
- карусель переключается через next / prev;
- кнопки карусели используют `data-action`;
- event log показывает действия пользователя;
- студент может объяснить `target`, `currentTarget`, bubbling и delegation.
