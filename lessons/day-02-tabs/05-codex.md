# Day 02: Codex

## Что можно отдать Codex

Codex можно попросить:

- создать папку `day-02-tabs`;
- создать `index.html`;
- создать `style.css`;
- создать пустой `script.js`;
- подготовить БЭМ-классы;
- сверстать 3 вкладки;
- добавить `data-tab` и `data-tab-panel`;
- оформить активные состояния через CSS;
- проверить мой код после самостоятельной реализации;
- объяснить ошибку простыми словами;
- предложить рефакторинг после того, как я сам написал решение;
- подготовить commit message.

---

## Что я пишу сам

Самостоятельно пишу:

- поиск кнопок через `querySelectorAll`;
- поиск панелей через `querySelectorAll`;
- `forEach` по кнопкам;
- обработчик клика;
- получение `event.currentTarget`;
- чтение `dataset.tab`;
- удаление active-классов;
- добавление active-классов;
- поиск нужной панели;
- проверку результата в браузере.

---

## Что нельзя отдавать Codex

Codex нельзя просить:

- полностью написать `script.js`;
- написать обработчики кликов за меня;
- написать всю логику переключения вкладок;
- исправить код без объяснения причины;
- коммитить код, который я не могу объяснить.
