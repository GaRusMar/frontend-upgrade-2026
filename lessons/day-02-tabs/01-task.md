# Day 02: Tabs — вкладки и переключение контента

## Цель урока

Научиться переключать вкладки на странице:

- находить несколько элементов через JavaScript;
- обрабатывать клики по кнопкам;
- удалять активный класс у старых элементов;
- добавлять активный класс нужному элементу;
- показывать нужный контент;
- скрывать лишний контент;
- связывать кнопку и блок контента через `data-*` атрибуты.

---

## Что делаем

Создаём простой блок вкладок:

```text
[HTML] [CSS] [JavaScript]

HTML content
```

Поведение:

- при клике на вкладку она становится активной;
- предыдущая активная вкладка перестаёт быть активной;
- показывается только контент выбранной вкладки;
- остальные панели скрываются.

---

## Какие темы закрепляем

- `querySelectorAll`
- `forEach`
- `addEventListener`
- `classList.add`
- `classList.remove`
- `data-*` атрибуты
- `dataset`
- `event.currentTarget`
- связь “кнопка → контент”
- работа с несколькими DOM-элементами

## HTML/CSS-классы

Используем БЭМ:

```text
tabs
tabs__nav
tabs__button
tabs__button--active
tabs__content
tabs__panel
tabs__panel--active
```

---

## Критерий готовности

Задача выполнена, если:

- на странице есть минимум 3 вкладки;
- при клике активной становится только выбранная вкладка;
- активная вкладка получает класс `tabs__button--active`;
- активная панель получает класс `tabs__panel--active`;
- видна только одна панель;
- связь кнопки и панели сделана через `data-tab`;
- в консоли браузера нет ошибок;
- JavaScript-логика написана самостоятельно.
