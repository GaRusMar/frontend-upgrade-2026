## Ответы

1. Объект — структура данных, которая хранит значения по именам.
2. Свойство — пара ключ/значение внутри объекта.
3. Ключ — имя свойства.
4. Через фигурные скобки: `const product = { name: 'Soup' }`.
5. `product.name`.
6. Когда имя свойства хранится в переменной или содержитспециальные символы.
7. `product.price = 18`.
8. `product.currency = '€'`.
9. `delete product.currency`.
10. Через `for...in` или методы `Object.keys/values/entries`.
11. Возвращает массив ключей объекта.
12. Возвращает массив значений объекта.
13. Возвращает массив пар `[key, value]`.
14. Несколько переменных могут указывать на один и тот же объект.
15. Потому что копируется ссылка, а не сам объект.
16. Через spread: `{ ...product }`.
17. Метод — функция внутри объекта.
18. `this` обычно указывает на объект, который вызвал метод.
19. У стрелочной функции нет собственного `this`.
20. Это способ достать свойства объекта в переменные.
21. `const { name: productName } = product`.
22. `const { description = 'No description' } = product`.
23. Это способ достать элементы массива в переменные.
24. Получится `[object Object]`; нужно выводить конкретные свойства.
25. В товарах, корзине, заказе, покупателе, настройках, ответах API.
