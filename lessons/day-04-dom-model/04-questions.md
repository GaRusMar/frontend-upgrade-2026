## Ответы

1. DOM — объектная модель документа, живая структура страницы в браузере.
2. `document` — главный объект страницы, через который можно работать с DOM.
3. Ищет первый элемент по CSS-селектору.
4. `null`.
5. Ищет все элементы по CSS-селектору.
6. `textContent` меняет текст, `innerHTML` вставляет HTML.
7. Через него можно небезопасно вставить пользовательский HTML.
8. Создаёт новый DOM-элемент.
9. Потому что его ещё нужно вставить в DOM через `append` или другой метод.
10. Добавляет элемент в конец контейнера.
11. Добавляет элемент в начало контейнера.
12. Заменяет содержимое контейнера; без аргументов очищает его.
13. Запись в HTML, например `src`, `alt`, `data-id`.
14. Свойство DOM-объекта, например `textContent`, `src`, `hidden`.
15. Объект для работы с `data-*` атрибутами.
16. `card.dataset.productId = product.id`.
17. Добавляет CSS-класс.
18. Удаляет CSS-класс.
19. Переключает CSS-класс.
20. Проверяет наличие CSS-класса.
21. `classList` не перезаписывает случайно все классы.
22. Функция, которая превращает данные в интерфейс.
23. Чтобы элементы не дублировались.
24. Состояние интерфейса, когда данных нет.
25. Во вкладке DevTools Elements.
26. JavaScript меняет живой DOM, а не файл на диске.
27. В карточках товаров, меню категорий, фильтрах, корзине, модалке и форме.
