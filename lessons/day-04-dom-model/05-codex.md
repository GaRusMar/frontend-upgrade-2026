# Day 04 — Codex

## Что можно отдать Codex

Codex можно попросить:

- создать структуру папки урока;
- создать `index.html`;
- создать `style.css`;
- создать пустой `script.js` с TODO;
- проверить мой код после самостоятельной реализации;
- объяснить ошибку из консоли;
- объяснить, почему `querySelector` вернул `null`;
- проверить, не дублируются ли карточки после render;
- предложить рефакторинг после того, как я сам написал решение.

## Что студент пишет сам

Самостоятельно пишу:

- массив `products`;
- массив `categories`;
- поиск DOM-контейнеров;
- функцию `formatPrice`;
- функцию `createProductCard`;
- создание DOM-элементов через `createElement`;
- заполнение текста через `textContent`;
- добавление классов через `classList`;
- запись `dataset.productId`;
- функцию `renderProducts`;
- очистку контейнера перед render;
- empty state;
- функцию `renderCategories`;
- featured product render;
- обработчики кнопок render / clear.

## Что нельзя отдавать Codex

Codex нельзя просить:

- полностью написать `script.js`;
- написать render-функции за меня;
- написать всю генерацию карточек за меня;
- исправить код без объяснения причины;
- заменить моё понимание готовым решением;
- коммитить код, который я не могу объяснить.
